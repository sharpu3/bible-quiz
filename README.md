# 성경 퀴즈 플래시 카드

모바일 브라우저에서 동작하는 성경 말씀 퀴즈 플래시 카드 앱입니다.

🔗 **배포 주소**: https://sharpu3.github.io/bible-quiz/

---

## 기능

### 기본 기능
- 플래시 카드 (문제 → 정답 순서로 표시)
- 매 시작마다 문제 순서 랜덤 셔플
- 원본 문제 번호 표시 (Q1~Qn)
- 진행률 표시 바
- 화면 어디나 탭하면 다음으로 이동 (네비게이션 버튼 제외)

### 네비게이션
- 화면 탭 또는 **다음** 버튼: 다음 단계 (문제 → 정답 → 다음 문제)
- **이전** 버튼: 정답 화면 → 해당 문제로, 문제 화면 → 이전 문제로

### 자동 넘기기 (자동 버튼)
- 활성화 시 문제/정답 표시 시간 설정 가능 (기본: 문제 10초, 정답 5초, 범위 문제 3~60초 / 정답 2~60초)
- 카운트다운 진행 바 표시
- 다시 누르면 중단
- 수동으로 넘기면 새 카드부터 카운트 리셋

### 읽기 TTS (🔊 버튼)
- 한국어 음성으로 카드 내용 자동 읽기
- 카드가 바뀔 때마다 이전 읽기 중단 후 새 카드 읽기 시작
- 다시 누르면 중단

### 자동 + 읽기 동시 사용
| 모드 | 동작 |
|---|---|
| 자동만 | 설정한 시간 후 자동 넘김 + 카운트다운 바 |
| 🔊만 | 읽어주고 수동으로 넘김 |
| 자동 + 🔊 | 읽기 완료 후 2초 카운트다운 바 표시 → 자동 넘김 |

### 백그라운드 처리
- 다른 탭으로 전환하거나 앱을 백그라운드로 보내면 TTS와 타이머 즉시 중단
- 다시 이 페이지로 돌아오면 현재 카드에서 현재 모드 그대로 재시작
- 화면 켜짐 유지(Wake Lock)는 이 페이지를 보는 동안에만 동작

## 문제 추가/수정

`quiz.js` 파일만 수정하면 됩니다. 문제 수는 자동으로 반영됩니다.

```js
const quizData = [
  {
    number: 1,               // 원본 문제 번호
    question: "문제 내용",
    answer: "정답 (또는 보조 정답)"
  },
  // ...
];
```

- 문제 수 변경 시 시작 화면, 완료 화면 문구 자동 업데이트
- `(또는 ...)` 형식으로 작성하면 정답 화면에서 보조 정답이 작은 글씨로 분리 표시

## 화면 구성

```
[시작 화면]
  시작하기 버튼

[퀴즈 화면]
  진행률 바 / 문제 번호
  문제 카드 → 정답 카드 (화면 탭으로 이동)
  [◀ 이전] [자동] [🔊] [다음 ▶]

[완료 화면]
  처음부터 다시 버튼
```

## 기술 스택

- HTML / CSS / JavaScript (순수 정적 파일, 프레임워크 없음)
- Web Speech API — 브라우저 내장 한국어 TTS (iOS onerror/resume 방어 포함)
- Screen Wake Lock API — 화면 꺼짐 방지 (포그라운드 한정)
- GitHub Pages 무료 배포

## 모바일 호환성

- `viewport-fit=cover` + `env(safe-area-inset-bottom)` — 노치/다이나믹 아일랜드 대응
- `100dvh` fallback — iOS Safari 주소창 높이 변동 대응
- 핀치줌 허용 — 노인 사용자 글자 확대 가능

## 대상

어르신 사용자를 고려하여 큰 글자(문제 24px, 정답 32px)와 터치하기 쉬운 버튼(높이 64px)으로 설계되었습니다.
