# 성경 퀴즈 플래시 카드

모바일 브라우저에서 동작하는 성경 말씀 퀴즈 플래시 카드 앱입니다.

🔗 **배포 주소**: https://sharpu3.github.io/bible-quiz/

---

## 기능

### 기본 기능
- 30문제 플래시 카드 (문제 → 정답 순서로 표시)
- 매 시작마다 문제 순서 랜덤 셔플
- 원본 문제 번호 표시 (Q1~Q30)
- 진행률 표시 바

### 네비게이션
- 카드 탭 또는 **다음** 버튼: 다음 단계로 이동 (문제 → 정답 → 다음 문제)
- **이전** 버튼: 정답 화면에서 누르면 해당 문제로, 문제 화면에서 누르면 이전 문제로

### 자동 넘기기
- **자동** 버튼으로 활성화/비활성화 토글
- 활성화 시 문제/정답 표시 시간 설정 가능 (기본: 문제 10초, 정답 5초)
- 카운트다운 진행 바 표시
- 수동으로 넘기면 새 카드부터 카운트 리셋

### 읽기 (TTS)
- **🔊** 버튼으로 한국어 음성 읽기 활성화/비활성화 토글
- 카드가 바뀔 때마다 내용 자동 읽기
- 수동으로 넘기면 현재 읽기 중단 후 새 카드 읽기 시작

### 자동 + 읽기 동시 사용
| 모드 | 동작 |
|---|---|
| 자동만 | 설정한 시간 후 자동 넘김 |
| 🔊만 | 읽어주고 수동으로 넘김 |
| 자동 + 🔊 | 읽기 완료 후 2초 대기 → 자동 넘김 (타이머 무시) |

## 화면 구성

```
[시작 화면]
  시작하기 버튼

[퀴즈 화면]
  진행률 바 / 문제 번호
  문제 카드 (탭하면 다음 단계)
  정답 카드 (탭하면 다음 문제)
  [◀ 이전] [자동] [🔊] [다음 ▶]

[완료 화면]
  처음부터 다시 버튼
```

## 기술 스택

- HTML / CSS / JavaScript (순수 정적 파일, 프레임워크 없음)
- Web Speech API (브라우저 내장 TTS, 별도 라이브러리 없음)
- GitHub Pages 무료 배포

## 대상

노인 사용자를 고려하여 큰 글자(문제 24px, 정답 32px)와 터치하기 쉬운 버튼(높이 64px)으로 설계되었습니다.
